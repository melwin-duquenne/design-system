### Semantiques

| Token | Role |
|---|---|
| `--color-text-primary` | Texte principal |
| `--color-text-secondary` | Texte secondaire |
| `--color-action-primary` | Actions principales (orange) |
| `--color-action-secondary` | Actions secondaires (bleu) |
| `--color-border-default` | Bordures par defaut |
| `--color-background-primary` | Fond principal |
| `--color-background-secondary` | Fond secondaire |

## Theme sombre

Le theme sombre est active via la classe `html.dark`. Les tokens semantiques sont automatiquement remappes :

| Token | Light | Dark |
|---|---|---|
| `--color-text-primary` | `neutral-900` | `white` |
| `--color-text-secondary` | `neutral-600` | `neutral-50` |
| `--color-background-primary` | `white` | `neutral-900` |
| `--color-background-secondary` | `neutral-50` | `neutral-600` |
| `--color-border-default` | `neutral-200` | `neutral-600` |
