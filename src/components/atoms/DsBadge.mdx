## Accessibilite

- Utilise un `<span>` avec un style `inline-flex`.
- Le contraste de couleur est assure par les tokens du Design System.
- Pour le badge `warning`, le texte est fonce pour garantir la lisibilite.

## Guidelines

| Do | Don't |
|---|---|
| Utiliser des textes courts (1-2 mots) | Mettre des phrases entieres dans un badge |
| Utiliser la variante semantique adaptee | Utiliser `error` pour un statut de succes |
| Utiliser `neutral` pour les informations non critiques | Surcharger l'interface de badges colores |
