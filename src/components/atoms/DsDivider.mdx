## Accessibilite

- Utilise `role="separator"` pour indiquer la separation aux technologies d'assistance.
- L'element HTML `<hr>` est semantiquement correct pour les separateurs.

## Guidelines

| Do | Don't |
|---|---|
| Utiliser pour separer des sections de contenu | Utiliser pour un espacement simple (utiliser les tokens de spacing) |
| Utiliser `vertical` entre des elements en ligne | Imbriquer des dividers |
