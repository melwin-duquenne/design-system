## Accessibilite

- Utilise l'element natif `<input type="text">`.
- La bordure change de couleur au focus pour indiquer l'etat actif.
- L'etat `disabled` empeche la saisie et change l'apparence.
- Associer un `<label>` via `DsInputGroup` pour l'accessibilite des formulaires.

## Guidelines

| Do | Don't |
|---|---|
| Toujours fournir un `placeholder` descriptif | Laisser le champ sans indication |
| Utiliser `DsInputGroup` pour ajouter un label | Utiliser `DsInput` seul dans un formulaire |
| Utiliser `disabled` pour les champs non editables | Cacher les champs avec du CSS |
