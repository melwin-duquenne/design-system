## Accessibilite

- Utilise `role="status"` pour indiquer le chargement aux technologies d'assistance.
- Ajouter un texte visuellement cache (sr-only) pour decrire l'etat de chargement dans le contexte parent.

## Guidelines

| Do | Don't |
|---|---|
| Utiliser `sm` pour les boutons et elements inline | Afficher un spinner sans contexte |
| Utiliser `lg` pour les zones de contenu principales | Laisser un spinner indefiniment sans timeout |
| Accompagner d'un message textuel si necessaire | Utiliser plusieurs spinners simultanement |
