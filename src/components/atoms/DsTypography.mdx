## Accessibilite

- Les balises HTML semantiques sont generees automatiquement : `h1`, `h2` pour les titres, `p` pour le body.
- Respecter la hierarchie des titres (ne pas sauter de niveau).

## Guidelines

| Do | Don't |
|---|---|
| Utiliser `h1` pour le titre principal de la page | Avoir plusieurs `h1` sur une meme page |
| Utiliser `weight` pour varier la graisse sans changer le niveau semantique | Utiliser `h2` pour le style visuel sans respecter la hierarchie |
| Utiliser `body-sm` pour les textes secondaires | Utiliser une taille de police personnalisee en dehors du systeme |
