# DsTokenShowcase

Section de presentation de design tokens. Combine un titre `DsTypography` et une `DsCard` pour encapsuler le contenu.

<Canvas of={DsTokenShowcaseStories.Default} />

## Utilisation

```vue
<DsTokenShowcase title="Couleurs">
  <!-- Swatches de couleurs -->
</DsTokenShowcase>
```

## Accessibilite

- Utilise la balise semantique `<section>`.
- Le titre est rendu en `<h2>`.

## Guidelines

- **Do** : Utiliser pour documenter visuellement les tokens du design system.
- **Don't** : Ne pas utiliser pour du contenu non lie aux tokens.
